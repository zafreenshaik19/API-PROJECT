# 🐶 Dog Image Generator

## About
This is a simple web application that displays a random dog image using the Dog CEO API.

## Features
- Displays a random dog image
- Fetches data using JavaScript
- Uses HTML, CSS and JavaScript
- Simple and responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API

## API
https://dog.ceo/api/breeds/image/random
